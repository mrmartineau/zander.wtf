title: Lab experiments + ZUI playground
date: 2026-07-17
---

A batch of new experiments on [lab.zander.wtf](https://lab.zander.wtf): a **keyboard-only navigation** demo, a **clock quiz** for practising telling the time (with relative-time questions), and a **procedural page** — an endless, self-generating page that builds itself as you scroll.

The procedural idea also landed in the [ZUI](https://github.com/mrmartineau/zui) docs as a **procedural playground**: an infinite page generated entirely from ZUI components. Alongside that, ZUI grew a VS Code extension with version-accurate completions, a Raycast extension for reference lookups, and a redesigned docs homepage.
