title: Launched fonts.zander.wtf
date: 2026-07-04
---

Launched [fonts.zander.wtf](https://fonts.zander.wtf), a curated showcase of free, high-quality typefaces. Every font is one I'd actually use — no endless scrolling through Google Fonts trying to separate the gems from the filler. Each typeface gets its own detail page with live previews, weight and OpenType feature toggles, and install options via [Fontsource](https://fontsource.org) or download.

It also ships a **font-picker agent skill**, so AI coding agents can pick fonts and pairings from the catalogue when scaffolding a new site — there's a [/skill/](https://fonts.zander.wtf/skill/) page explaining how to install it.
