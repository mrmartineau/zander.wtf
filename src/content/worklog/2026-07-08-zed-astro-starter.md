Put together [zed-astro-starter](https://github.com/mrmartineau/zed-astro-starter), an opinionated Astro starter ready to deploy to Cloudflare Workers. It pairs Astro 7 with [ZUI](https://github.com/mrmartineau/zui) for styling, [Vite+](https://viteplus.dev) as a unified toolchain for formatting, linting, type checking and testing, and GitHub Actions for deploys plus a supply-chain scan on every branch. Clone it, `pnpm install`, and you're building.
