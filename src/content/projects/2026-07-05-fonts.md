Whenever I start a new project I go hunting for font inspiration, and I've bookmarked countless typefaces I liked along the way. What I never had was a good list of my own — somewhere I could preview them properly, edit the sample text, and play with different font feature settings.

There are similar sites out there that I use and like, but none of them have exactly the fonts and features I want. So I built Free Fonts for myself, first and foremost — a curated collection of free web fonts I can (and do) use, all in one place. And as with anything I make, it's there for everyone else to try and use too.
