A growing collection of Raycast extensions — some published to the Raycast store, some personal-use only.

## Extensions

### Search npm Packages

Search and favourite npm packages directly from Raycast. 18k+ users on the Raycast store.

### GitHub Stars

Display and filter your recent GitHub stars. 3.3k+ users on the Raycast store.

### Bird

View your X/Twitter bookmarks and likes in Raycast using the `bird` CLI.

### Otter

View, search and add bookmarks to my Otter bookmarking project. Not yet published to the store.

### Code Notes Search

Search my code notes with Algolia. Not yet published to the store.
